DEFAULT PALETTE — MONOCHROME + GREEN ACCENT (LOCKED)
----------------------------------------------------
Founder decision (2026-07-07): "We're going with monochrome plus the green accent."

The canonical Horus look is mostly grayscale with dark forest green reserved for key interactive moments — not a fully green-tinted UI.

| Layer              | Treatment                                                                 |
|--------------------|---------------------------------------------------------------------------|
| Base               | Off-white / white backgrounds · black or near-black text · gray borders · gray secondary/disabled states |
| Structure          | Monochrome chrome everywhere by default — nav links, breadcrumbs, inputs, modals, footers stay neutral |
| Green accent       | Dark forest green ONLY on: primary CTA · active tab · selected checkbox/radio · toggle-on · Confirm button · focused input border (optional) |
| Error              | Red border + red helper text — never green                                |
| Stays black/gray   | Secondary buttons · ghost/tertiary links · inactive tabs · disabled states · body copy · dividers |

Rule of thumb: If it's not clickable or "active," it stays monochrome. Green is punctuation, not wallpaper.

PURE MONOCHROME (no green)
- Background: off-white / very light gray
- Accent: solid black only
- Buttons: primary = black fill + white text; secondary = white + black border
- Use when: stripped-down contexts, print, or zero-color requirements

CONCRETE + GREEN (alternate)
- Background: light neutral "concrete" gray
- Primary accent: dark forest green on buttons, tabs, controls, CTAs
- Feel: greener overall environment than founder default

DEEP SPEC (alternate)
- Same green accent family on warmer cream / parchment backgrounds
- Borders: more pronounced green-accent borders on containers
- Use when: marketing that needs warmth without losing institutional tone

UI COMPONENT PATTERNS (FROM MOCKUPS)
------------------------------------
| Component       | Pattern                                                                 |
|-----------------|-------------------------------------------------------------------------|
| Top nav         | Logo left · text links center/right · primary CTA (e.g. "Book a demo") |
| Buttons         | Primary (solid accent) · Secondary (outlined) · Ghost/Tertiary · Disabled |
| Text input      | Default · Focused (accent border) · Error (red border + message)       |
| Dropdown        | Closed select · open menu with account-type options                      |
| Checkbox + radio| Unselected outline · selected = accent fill                            |
| Toggle          | Off (gray) · On (accent)                                                 |
| Tabs            | Text tabs · active = accent underline                                    |
| Breadcrumb      | Home > Products > … hierarchy                                            |
| Badges / tags   | Pill shapes — e.g. "Enterprise", "SOC 2"                                 |
| Search bar      | Full-width field · magnifier · optional ⌘ K hint                         |
| Modal           | Title · body copy · Cancel (secondary) + Confirm (primary)               |
| Footer links    | Privacy · Terms · Security · Contact · copyright line                    |
